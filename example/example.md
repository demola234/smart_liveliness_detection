# smart_liveliness_detection Example

A full working example is in [`lib/main.dart`](lib/main.dart).

It demonstrates:
- Default liveness detection
- Custom theme and challenge configuration
- Screen flash anti-spoofing
- Face quality scoring
- 3D depth detection (iOS TrueDepth)
- Biometric template enroll and verify
- Voice guidance
- Futuristic UI styles and the runtime style picker
- Challenge hint animations
